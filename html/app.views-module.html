index(*args, **kwargs)

source code 

This function had been connected with flask's router by decorating itself with 'app.route'. While decorated flask tranlsates all parameters from url (with syntax: '/some/url/<parameter>') into function arguments (right here function does not have any paramaters) while calling. Also it decorated with templating decorator, which do dirty work (call render_template with putting into arguments from dictionary if nessesary and other).

index() just render main page with posts on them and login form, and, if it's POST request right now, validating form and checking user's password.

Returns: dict
return a python's dictionary (map structure) with parameters in dict items, which would be translated into jinja templater.
Decorators:
  • @app.route("/", methods= ["GET", "POST"])
  • @app.route("/index", methods= ["GET", "POST"])
  • @app.route("/posts", methods= ["GET", "POST"])
  • @templated("index.html")
